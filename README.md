<br/>

**Step 1** — In your GitHub profile repo (`ParthKaretiya/ParthKaretiya`), create this file:

**`.github/workflows/snake.yml`**

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"   # runs every day at midnight
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: ParthKaretiya
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**Step 2** — Go to **Actions → Generate Snake Animation → Run workflow**

The snake will appear on your profile within a minute! ✅
